A simple and easy **NodeJS** game.

_My entry for **[Kajam 2022](https://kajam.replit.com/)**_



### How to Play
1. Weed out all the rotten ghost peppers with smudges
2. Prevent the timer from running out
3. Finish all 5 levels
4. **You win!!**
